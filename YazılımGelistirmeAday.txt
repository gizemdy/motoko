Aşağıdaki sütunları içeren tabloları mevcut değilse oluşturan SQL komutlarını yazınız.

Çalışanlar Tablosu
    • id: Birincil anahtar, tamsayı, otomatik artan.
    • ad: Çalışanın adı, metin.
    • soyad: Çalışanın soyadı, metin.
    • pozisyon: Çalışanın pozisyonu, metin.
    • maas: Çalışanın maaşı, ondalık.
    • departman_id: Çalışanın bağlı olduğu departmanın ID'si (yabancı anahtar). 


Departmanlar Tablosu
    • id: Birincil anahtar, tamsayı, otomatik artan.
    • departman_adi: Departmanın adı, metin.

CREATE TABLE calisanlar
	    id(10) INT,PRIMARY, AUTO IN..,
	     calisan_adi(255) TEXT,
             calisan_soyadi(255) TEXT,
	     pozisyon(255) TEXT,
	     maas(10) DECL ?,
	     departman_id(10) FOREIGN KEY
;
CREATE TABLE departmanlar
	    id(10) INT,PRIMARY, AUTO IN..,
	     departman_adi(255) TEXT
;
